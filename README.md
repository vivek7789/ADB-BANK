# ADB-BANK
banking system by using c-language

## Description

ADB-BANK is a comprehensive banking system project built with C++ that simulates core banking operations and functionality.

## Features

- Account management
- Transaction processing
- User authentication
- Balance inquiries
- Deposit and withdrawal operations

## Prerequisites

Before you begin, ensure you have the following installed:
- C++ compiler (GCC 7.0+ or Clang 5.0+)
- Make (optional, for build automation)
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/vivek7789/ADB-BANK.git
cd ADB-BANK
```

2. Compile the project:
```bash
g++ -o bank main.cpp
```

Or if you have a Makefile:
```bash
make
```

## Usage

Run the program:
```bash
./bank
```

Follow the on-screen menu to:
- Create a new account
- Login to your account
- Perform transactions
- Check balance
- Exit the system
